# Cryptoverse - Explore the World of Cryptocurrency

## Introduction
A Crypto Currency App and is made with React, Redux and Crypto rapid-api. On here, you can able to search the crypto currency, see the Crypto currency in detail, see the value statistics, market cap and supply with time filter.

Technical Stack-
Frontend: React, Redux for state management and Ant design for css
Backend: Crypto rapid-api

Setup & Deployment-
React.js: The frontend app is deployable on platforms like Vercel, Netlify or a custom server.
Backend: Rapid-api.

Environment- Ensure environment variables are correctly set for CRYPTO_RAPIDAPI_HOST, RAPIDAPI_KEY, CRYPTO_API_URL.
